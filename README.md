# VK Trial Auto Supporter

お試し申請の対応を自動化するプラグインです

## 依存プラグイン

- UpdraftPlus
- Contact Form 7
- User Role Editor

## 初期値を変更したい場合
1. 変更を加える
2. バックアップを取る
3. 生成したバックアップファイルのnonceを見てプラグインのnonceを書き換える

## インターバルを変更したい場合
イベントとインターバルの連携はプラグイン有効化時に設定される

よって
1. プラグインを一度無効化する
2. インターバルを変更する
3. プラグインを有効化する
すると指定した時間からイベントを実行することが出来ます

WP Crontrolプラグインを利用して「ツール > Cron イベント」からイベントを確認することが出来ます
